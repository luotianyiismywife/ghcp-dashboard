### 基础信息

```json
    "name": "ghcp-dashboard",
    "displayName": "GitHub Copilot Insights Dashboard",

    "license": "MIT"
}
```

### 引擎要求

```json
"engines": { "vscode": "^1.95.0" }
```

需要 VS Code 1.95+ 的原因：
- `vscode.lm.selectChatModels()` API
- `vscode.lm.tools` API
- `vscode.authentication.getAccounts()` API
- `vscode.l10n.t()` API

### 关键新字段：`"l10n"`

```json
"l10n": "./l10n"
```

**必须配置**，否则 VS Code 不会加载 `l10n/bundle.l10n.*.json` 翻译包，`vscode.l10n.t()` 始终返回原文。

## 贡献点详解

### commands（3 个）

| 命令 ID | 标题 | 图标 | 用途 |
|---------|------|------|------|
| `ghcpDashboard.open` | GHCP: Open Copilot Insights Dashboard | `$(github)` | 打开完整仪表板面板 |
| `ghcpDashboard.refresh` | GHCP: Refresh Account Data | `$(refresh)` | 刷新所有数据 |
| `ghcpDashboard.switchAccount` | GHCP: Switch GitHub Account | — | 显示账户切换 QuickPick |

### viewsContainers（1 个）

在活动栏添加 GitHub 图标容器：
```json
"ghcp-dashboard": { "title": "GitHub Copilot Insights Dashboard", "icon": "$(github)" }
```

### views（1 个）

侧边栏 WebView 视图：
```json
"ghcpDashboard.sidebarView": { "type": "webview", "name": "Account Overview" }
```

### menus（1 个）

在侧边栏标题栏添加刷新按钮：
```json
"view/title": [{ "command": "ghcpDashboard.refresh", "when": "view == ghcpDashboard.sidebarView" }]
```

## 依赖

### devDependencies

| 包 | 用途 |
|---|------|
| `@types/vscode` ^1.95.0 | VS Code API 类型定义 |
| `@vscode/vsce` ^2.22.0 | VSIX 打包工具 |

**无运行时依赖** —— 整个扩展纯原生 JS，不依赖任何 npm 包。

## 脚本

```json
{
    "package": "npx @vscode/vsce package --allow-missing-repository",
    "package:dev": "node scripts/build-dev.js"
}
```

注意：`scripts/build-dev.js` 文件在仓库中**不存在**（引用的路径实际不存在）。
