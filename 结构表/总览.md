# 项目总览

## 一句话

在 VS Code 内部查看 GitHub Copilot 账户、AI 用量统计、聊天会话历史、MCP 服务器连接和语言模型——一站式面板。

## 元信息

| 项目 | 值 |
|------|-----|
| 名称 | `ghcp-dashboard` |
| 版本 | 2.0.4 |
| 发布者 | `luotianyiismywife` |
| 原作者 | Jadhav Shubhamm（`shujadhav`） |
| 许可证 | MIT |
| 最低 VS Code | `^1.95.0` |

## 文件结构

```
ghcp-dashboard/
├── package.json                 # 扩展清单
├── package.nls.json             # 清单本地化（英）
├── package.nls.zh-cn.json       # 清单本地化（中）
├── .gitignore
├── LICENSE.txt
├── README.md
├── media/                       # 图标 (3 png)
├── l10n/                        # 翻译包
│   ├── bundle.l10n.json
│   └── bundle.l10n.zh-cn.json
├── src/                         # 源码
│   ├── extension.js             # 扩展入口
│   ├── accountDataFetcher.js    # 数据获取层
│   ├── dashboardPanel.js        # 仪表板面板
│   ├── sidebarProvider.js       # 侧边栏
│   └── trendsPanel.js           # 趋势分析
├── test/                        # 测试脚本 (8个)
├── .github/                     # Copilot 指令
└── 结构表/                      # 项目结构文档
```

## 数据流

```
VS Code API ──┐
              ├──→ AccountDataFetcher.getAllData() ──→ DashboardPanel / SidebarProvider
state.vscdb ──┘                        ↓
                                   data 对象
```
