# 项目总览

## 一句话

> 在 VS Code 内部查看 GitHub Copilot 账户、AI 用量统计、聊天会话历史、MCP 服务器连接和语言模型——一站式面板。

## 元信息

| 项目 | 值 |
|------|-----|
| 名称 | `ghcp-dashboard` |
| 显示名 | GitHub Copilot Insights Dashboard |
| 版本 | 2.0.4 |
| 发布者 | `luotianyiismywife` |

| 许可证 | MIT |
| 最低 VS Code | `^1.95.0` |
| 激活方式 | 无 `activationEvents`（延迟加载，命令触发时激活） |

## 项目目的

该扩展通过读取 VS Code 原生 API 和本地 SQLite 数据库（`state.vscdb`），将散落在 VS Code 各处的 Copilot 相关信息聚合到一个可视化界面中。**完全离线工作，不依赖任何外部 API。**

## 技术栈

- **运行时**: Node.js（VS Code 扩展宿主）
- **API**: VS Code Extension API
  - `vscode.authentication` — 账户信息
  - `vscode.lm` — 语言模型和工具
  - `vscode.extensions` — Copilot 安装状态
  - `vscode.workspace` — 配置和工作区
  - `vscode.l10n` — 国际化/本地化
  - `vscode.WebviewView` / `vscode.WebviewPanel` — UI
- **数据源**: VS Code 内部 SQLite 数据库
  - `全局: %APPDATA%/Code/User/globalStorage/state.vscdb`
  - `工作区: %APPDATA%/Code/User/workspaceStorage/*/state.vscdb`
- **UI**: WebView（HTML + CSS + Vanilla JS），无前端框架
- **i18n**: `vscode.l10n.t()` + `package.nls.json` + `l10n/bundle.l10n.json`

## 国际化（i18n）

| 文件 | 用途 |
|------|------|
| `package.nls.json` | 清单（package.json）的英文本地化 |
| `package.nls.zh-cn.json` | 清单的中文翻译 |
| `l10n/bundle.l10n.json` | 代码字符串的默认英文包 |
| `l10n/bundle.l10n.zh-cn.json` | 代码字符串的中文翻译 |
| `package.json` → `"l10n": "./l10n"` | 必须配置，告知 VS Code 翻译包位置 |

所有用户可见字符串通过 `vscode.l10n.t('...')` 包裹，VS Code 根据界面语言自动加载对应翻译包。
