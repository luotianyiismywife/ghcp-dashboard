# 趋势面板 — `trendsPanel.js`

## 概述

仪表板中 Trends & Reports 标签页的实现模块。提供周/月/30 天维度的使用趋势分析，带对比期和导出功能。

## 导出函数

| 函数 | 返回 | 说明 |
|------|------|------|
| `getTrendsTabHTML(data, aiStatsEnabled)` | HTML 字符串 | 趋势标签页的完整 HTML |
| `getTrendsJS()` | JS 字符串 | 趋势页面客户端 JavaScript（图表渲染、周期切换、导出） |
| `getTrendsCSS()` | CSS 字符串 | 趋势页面的样式 |

## 功能特性

- **周期选择**: Week / Month / 30 Days
- **自动对比**: 当前周期 vs 上一周期（例如本周 vs 上周）
- **图表类型**:
  - 活跃天数柱状图
  - 字符数对比图（AI vs Typed）
  - AI Rate 折线图
  - 建议与编辑数图表
- **对比表格**: 各指标的数值、变化量和变化百分比
- **导出**: CSV / Markdown 格式导出

## 集成方式

由 `dashboardPanel.js` 在构建仪表板时调用：

```javascript
const { getTrendsTabHTML, getTrendsJS, getTrendsCSS } = require('./trendsPanel');

// 在 _getHtml() 中:
const trendsHtml = getTrendsTabHTML(data, aiStatsEnabled);
const trendsJs = getTrendsJS();
const trendsCss = getTrendsCSS();
```

注意：Trends 标签页当前在 UI 中**未作为独立 Tab 显示**，其内容通过 JS 嵌入到 AI Stats 标签页内。
