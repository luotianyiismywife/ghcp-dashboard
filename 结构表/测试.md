# 测试文件

对应目录：`test/`

## 概述

8 个独立的 Node.js 测试脚本，可在 VS Code 外部直接运行（`node test/test_*.js`）。不依赖 `vscode` 模块，直接读取文件系统。

## 测试文件清单

| 文件 | 命令 | 说明 |
|------|------|------|
| `test/test_verify.js` | `node test/test_verify.js` | 语法/结构/功能综合验证 |
| `test/test_ai_stats.js` | `node test/test_ai_stats.js` | AI Stats 数据格式和时间范围验证 |
| `test/test_aistats_diagnostics.js` | `node test/test_aistats_diagnostics.js` | AI Stats 诊断分析 |
| `test/test_fetcher_parity.js` | `node test/test_fetcher_parity.js` | 数据获取器一致性测试 |
| `test/test_heatmap.js` | `node test/test_heatmap.js` | 热力图数据验证 |
| `test/test_late_may_probe.js` | `node test/test_late_may_probe.js` | 5 月数据探测 |
| `test/test_may_28_31_raw_scan.js` | `node test/test_may_28_31_raw_scan.js` | 5 月 28-31 日原始扫描 |
| `test/test_may_workspace_inspect.js` | `node test/test_may_workspace_inspect.js` | 工作区检查 |

## 测试特点

- 无第三方依赖，使用 Node.js 内置 `fs`、`path` 模块
- 离线运行，不依赖 VS Code 或网络
- 复用 `accountDataFetcher.js` 的数据读取策略（二进制扫描 + 正则）
