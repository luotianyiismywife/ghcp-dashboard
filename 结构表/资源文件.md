# 资源文件

## 图标 — `media/`

| 文件 | 用途 |
|------|------|
| `GHCP_dash_icon_circle.png` | 仪表板面板图标、扩展图标 |
| `GHCP_dash_icon_sjtc.png` | 备用图标（SJTC 变体） |
| `GHCP_dash_icon_sjtc_1.png` | 备用图标（SJTC 变体 2） |

图标在以下位置使用：
- `package.json` 的 `icon` 字段
- `DashboardPanel.createOrShow()` 中的 `panel.iconPath`

## 工作区配置 — `ghcp-dashboard.code-workspace`

```json
{
    "folders": [{ "path": "." }],
    "settings": {
        "editor.insertSpaces": false,
        "editor.tabSize": 4
    }
}
```

- 使用 **Tab 缩进**（`insertSpaces: false`）
- Tab 宽度 **4 空格**
